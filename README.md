## 已確認的重要里程碑

- **2025-09-07**：漲停板走失後尋回。
- **2025-09-18**：搬到新店屈尺生活。
- **2026 年 1 月**：經進一步檢查後診斷 SRMA，開始類固醇治療。
- **2026-05-17**：依醫師指示停止類固醇。
- **2026 年 7 月**：精神與活動力良好，重新在屈尺公園探索與奔跑。
