## 已確認的重要里程碑

- **2025-09-07**：漲停板走失後尋回。
- **2025-09-18**：搬到新店屈尺生活。
- **2025-10-26**：媽媽抱起漲停板時，他第一次叫痛；家人當時先懷疑肚子不舒服。
- **2025-12-07**：永康動物醫院醫師建議進一步做 MRI。
- **2025-12-31**：到台大動物醫院進一步評估並安排檢查。
- **2026-01-05**：完成 MRI、腦脊髓液（CSF）與相關檢查。
- **2026-01-08**：開始每天服用六顆類固醇。
- **2026-05-17**：依醫師指示停止類固醇。
- **2026 年 7 月**：精神與活動力良好，重新在屈尺公園探索與奔跑。
- **2026-07-26**：預定開始每週日連載《那一年，我生病了》。

## 寫作設定

- 漲停板以第一人稱「我」說故事。
- 第三人稱統一使用「他」，不用「牠」。
- 家庭稱呼使用爸爸、媽媽、哥哥，不使用主人。
- 依照當時真正知道的資訊寫，不用後來的答案改寫當時心情。
- 不刻意煽情，但一定真誠。
