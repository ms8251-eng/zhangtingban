# 漲停板專案規劃

## 專案目的

這個專案不只是疾病紀錄，也是一份家庭共同陪伴漲停板走過生病、治療與康復的生命故事。

## 內容主軸

### 1. 生命故事

- 生病前的個性與生活
- 走失、搬家與屈尺新生活
- 發病與尋找病因
- 最害怕的日子
- 確診 SRMA
- 長期治療與家庭照護
- 停藥、觀察與重新奔跑

### 2. 醫療紀錄

- 就診時間軸
- MRI、CSF、X 光與血液檢查摘要
- 類固醇劑量變化
- 回診紀錄
- 停藥後觀察

### 3. 漲停板探險日記

- 屈尺公園
- 溪邊探索
- 山居日常
- 散步、食物與柴犬個性

### 4. 社群內容

- Threads 貼文
- LINE 圖文
- 照片與影片索引
- 未來可延伸為書籍、月曆或完整影像故事

## 公開與私人資料分流

本 GitHub Repository 目前為公開狀態，因此只放適合公開分享的內容。

不公開的資料包括：

- 家庭個人資料
- 完整醫療帳單與費用
- 含地址、電話、身分資料的單據
- 未經遮蔽的醫院文件

私人資料應另外保存在個人電腦、雲端硬碟或私人 Repository。
