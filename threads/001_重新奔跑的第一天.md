# Threads 001｜重新奔跑的第一天

2. 漲停板在溪邊探索的照片

## 貼文正文

半年前，

我因為 **SRMA（類固醇反應性腦膜炎動脈炎）** 生病。

那時候，
連低頭都會痛，
更別說奔跑了。

爸爸媽媽陪著我，
一起看醫生、做檢查、吃藥，
走過了將近半年的治療。

今天，

我又回到了最喜歡的屈尺公園。

走進溪邊，
聞聞青草，
看看水裡的倒影，
一步一步探索熟悉又新鮮的世界。

對很多人來說，
這只是柴犬散步的一天。

但對我們一家人來說，

**這是重新找回健康的一天。**

🐕 我是漲停板。

以後，我會分享我的探險日記、
我的康復故事，
還有屈尺山居的每一天。

希望把這份得來不易的快樂，也分享給大家。❤️

📍新北・屈尺公園
